Mesour\Selection

Required

Server:

  • PHP >= 5.4
  • Mesour\Components
  • Mesour\DropDown

Client:

  • Nothing required

Install

With composer. Go to getcomposer.org.

"require": {
    "mesour\table": "dev-master"
}

Or go to GIT repository

Features

Init:

$selection = new \Mesour\UI\Selection('test');

$items = array(
    1 => 'active',
    2 => 'inactive',
    3 => 'inactive',
    4 => 'active',
    5 => 'inactive',
);

$selection->setItems($items);

$selection->addStatus('active', 'Active');

$selection->addStatus('inactive', 'Inactive');

$selection->render();

foreach($items as $id => $status) {
    $selection->renderItem($id);
    echo " ID: $id, Status: $status";
    echo "<br>";
}

Using JS API


Getting values

mesour.selection.getValues('test')

//example output: Object {1: true, 2: false, 3: true, 4: false, 5: false}
//keys from given items

Getting items for selection with name "test"

mesour.selection.getItems('test'); //returns array with html nodes

Getting main checkbox HTML node

mesour.selection.getMainCheckbox('test')
